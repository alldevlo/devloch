## Mini checklist de review visuelle (desktop)

- H1 visible immédiatement dans le hero
- Hero media / bannière visible et bien cadré(e)
- CTA principal et sticky CTA pointent vers le formulaire (#contact-form)
- Sticky subnav visible pendant le scroll et navigation par sections fonctionnelle
- Bloc témoignage premium visible (photo + attribution)
- Formulaire HubSpot inline visible en bas de page
- Aucun bloc “source / audit / transparence” visible dans le flux public
